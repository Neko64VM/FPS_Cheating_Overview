# 概要・定義
本ドキュメントではチートを「FPSゲームにおいて開発者が想定していない不正な外部のソフトウェアやディバイスといったものを使用して他のプレイヤーに対し不正に優位に立つこと」と定義し、それらを使用するプレイヤーのことを「チーター (Cheater)」という。  
言わずもがな、対人FPSゲームの誕生から現代に至るまで、正規のゲーマーに最も嫌われる行為の1つである。
